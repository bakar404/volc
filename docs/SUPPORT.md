# Support

VolC is a small open-source project. Please open a GitHub issue for bugs, app support requests, or compatibility reports.

Include:

- macOS version
- Xcode version, if building from source
- VolC commit or release version
- target app, such as Spotify, Music, or Chrome
- what you expected
- what happened instead
- any message shown in the VolC popover

For Chrome issues, also say whether **View > Developer > Allow JavaScript from Apple Events** is enabled.
