# Code of Conduct

VolC follows the spirit of the [Contributor Covenant](https://www.contributor-covenant.org/) and expects contributors to keep the project welcoming, practical, and respectful.

## Our Standards

Examples of behavior that helps the project:

- using welcoming and inclusive language
- respecting different skill levels and macOS setups
- giving constructive feedback
- focusing disagreement on technical tradeoffs, not people
- being honest about limitations and risks

Examples of unacceptable behavior:

- harassment, insults, or personal attacks
- sexualized language or imagery
- trolling or sustained disruption
- publishing private information without permission
- dismissing safety concerns around permissions, automation, or audio capture

## Scope

This Code of Conduct applies in:

- GitHub issues
- pull requests
- discussions
- release comments
- any project-related community space

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or block users when behavior is harmful to the project or community.

Security issues should be reported through GitHub private vulnerability reporting, not public issues.
